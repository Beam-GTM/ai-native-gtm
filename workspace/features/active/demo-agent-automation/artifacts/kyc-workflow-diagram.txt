┌─────────────────────────────────────────────────────────────────────────────────┐
│                            DECISION BRANCHES                                   │
└─────────────────────────────────────────────────────────────────────────────────┘

DECISION POINT: Document Completeness
├─ ALL DOCUMENTS PRESENT → Continue to Data Extraction
├─ MISSING DOCUMENTS → Request Resubmission
└─ INVALID FORMAT → Reject with Instructions

DECISION POINT: Risk Assessment
├─ LOW RISK + ALL CHECKS PASSED → Auto Approve
├─ MEDIUM RISK → Manual Review Required
├─ HIGH RISK → Enhanced Due Diligence
└─ CRITICAL RISK/SANCTIONS MATCH → Auto Reject

DECISION POINT: Verification Results
├─ DOCUMENTS AUTHENTIC + IDENTITY VERIFIED → Continue
├─ FRAUD INDICATORS → Auto Reject
└─ VERIFICATION FAILED → Manual Review

DECISION POINT: Final Decision
├─ AUTO APPROVE → Send Approval Notification
├─ MANUAL REVIEW → Escalate to Compliance Team
└─ AUTO REJECT → Send Rejection with Reason
